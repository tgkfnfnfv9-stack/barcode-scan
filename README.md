# 会社バーコードスキャン｜株式会社小林機械

機械の銘板バーコードを iPhone で読み取り、社内ページ（機械在庫・在庫板）へ誘導する現場用ツールです。
読み取りエンジンは端末内で完結（オフライン動作）、単一の `index.html` だけで動きます。

**公開URL：** https://tgkfnfnfv9-stack.github.io/リポジトリ名/
（↑ 実際のURLに書き換えてください）

---

## 使い方

1. 上記URLを iPhone の Safari で開く（ホーム画面に追加すると便利）
2. あらかじめ Safari で `kkmt.co.jp/staff` にログインしておく
3. 「撮影して読み取る」で銘板のバーコードを撮影
4. 読み取ったコードが履歴に追加される（最新が一番上）
5. 各行の「機械在庫へ」または「在庫板へ」を押すと、該当ページが開く

### 主な機能

- 読み取り履歴が積み上がる（同じコードは重複せず先頭へ移動）
- 各コードから「機械在庫」「在庫板」の両方へ移動可能
- 個別削除（✕）／すべてリセット
- 「すべてコピー」で全バーコードを1行1コードの形式でコピー（事務所へ送る用）
- 履歴は端末内に保存（閉じても更新しても残る）

---

## 更新方法

`index.html` を新しい版で上書きコミットするだけで、数分後に公開URLへ自動反映されます（URLは変わりません）。

1. リポジトリの `index.html` を開く
2. 鉛筆（Edit）または「Upload files」で新しい版に差し替え
3. 「Commit changes（変更をコミット）」で保存

## 技術メモ

- 単一 HTML（CDN・ネット依存なし）。読み取りエンジンは ZXing-C++（WASM）を base64 で埋め込み
- 遷移先
  - 機械在庫：`https://www.kkmt.co.jp/staff/stocks/short_search?barcode=<コード>`
  - 在庫板：`https://www.kkmt.co.jp/staff/stock_boards/short_search?barcode=<コード>`
- 印字と符号化値が異なる機械は、`index.html` 内の `OVERRIDE_MAP` に登録して対応
